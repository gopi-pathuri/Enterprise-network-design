# Network Diagrams

This directory contains the visual representations of the enterprise network.

## Diagrams Included

1. **Logical Topology** (`topology-logical.png`)
   - Visualizes the three-tier architecture (Core, Distribution, Access).
   - Shows redundant links, VLANs, and OSPF connectivity.
   - *Simulated utilizing Cisco Packet Tracer style.*

2. **Physical Topology** (`topology-physical.png`)
   - Realistic view of the server rack and cabling.
   - detailing patch panel connections and device mounting.
   - *Simulated utilizing Cisco Packet Tracer Physical View.*

3. **CLI Verification** (`verification-cli.png`)
   - Screenshot of a terminal session on `Core-Switch-1`.
   - Proof of successful OSPF routing and network reachability.

## Tools Used
- Cisco Packet Tracer (Simulation)
- SecureCRT (Terminal Emulation)
